# KUET-BUS
Bus Schedule App For KUET Bus 
Used New [Android Jetpack](https://developer.android.com/jetpack/)

## Features
> Works in Offline

> One Tap to Update

> Sorted Bus Time for you

> Friendly UI (I guess :3 )

## Install

Right Click One the link and select save this link as
